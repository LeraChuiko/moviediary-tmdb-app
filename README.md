Group Project

🎬  Movie Diary (TMDB API)

1. On Homepage, fetch and display popular movies from the TMDB API.
2. Display each movie’s image, title, and info in a card.
3. “Add to favourites” button stores the movie as an object in an array inside localStorage.

4. Add a search bar; on submit, show results/feedback in a dialog.
5. Journal page lists favourite movies from localStorage, showing image, title, info.
6. Allow adding notes to each movie; persist the note on the same object in localStorage.
